stupid-lang Console Variables Conditions How to install

How to install:

For the coolest (no) programming language, you need to install the Nim programming language on your PC. Next, you must download the main.nim file and open the console. Next, you must create the test.stupid file. After that, you must enter this command into the console: nim c .\main.nim

After that you can open the compiled file.

Console:

clog() - Outputs a string, value, or variable.

clog("Hello, world!");

cin() - Allows the user to enter a string or value into a variable or empty.

var namevar; cin(namevar);

Variables:

To create a variable you need to write:

var namevar;

But if you want to immediately initialize your variable, write:

var namevar = "Mike";

To remove a variable you need to write:

remVar(namevar);

Conditions:

Conditions example:

var name;
cin(name);
if(name == "Mike");
clog("Hello developer!")
end;
if(name != "Mike");
clog("Hello user!")
end;
remVar(name);
